# DRAVIDA — Sacred Permanence Landing Page

An elegant, immersive, and content-rich landing page dedicated to exploring the architectural brilliance, petrified prayers, and historic legacy of South India's Dravidian temple complexes. This frontend project highlights the temporal evolution of temple architecture across major historical dynasties using modern web design aesthetics.

## 🌟 Features

*   **Immersive Hero Section:** Features striking, high-contrast imagery of ancient Dravidian temple corridors overlaid with deep, narrative-driven typography.
*   **The Chisel's Devotion Section:** A dedicated spotlight on structural craftsmanship, highlighting mathematical symmetry (Shilpa Shastras) and intricate high-relief carving techniques using clean iconography.
*   **Iconography & Symbolism Breakdown:** An interactive or structurally defined breakdown of cosmic forms (like the Tandava Cycle of Nataraja) and mythic/floral motifs (Yalis, Makaras, and Lotuses).
*   **Chronicles of Stone (Temporal Journey):** A clean chronological timeline tracking the evolution of Dravidian architecture across four distinct eras:
    *   **The Rock-Cut Dawn (600–750 CE):** Monolithic structures of the Pallava Dynasty (e.g., Mahabalipuram).
    *   **The Imperial Ascent (850–1200 CE):** Massive gopurams and bronze casting masterworks of the Chola Golden Age.
    *   **The Hoysala Intricacy (1000–1350 CE):** Soapstone plasticity and distinctive stellate (star-shaped) temple footprints.
    *   **The Vijayanagara Grandeur (1350–1565 CE):** Sprawling medieval complexes and heavily ornamented mandapas (e.g., Hampi).
*   **Visual Archive:** A highly responsive grid-based gallery showcasing curated architectural elements, sculptures, and historical ruins.
*   **Newsletter Subscription:** A minimalist conversion form inviting users to subscribe for monthly archaeological updates and preservation alerts.

## 🛠️ Tech Stack & Structure

*   **HTML5:** Structured semantic markup utilizing sections, headings, and data anchors for smooth timeline navigation.
*   **CSS3:** Custom layouts utilizing modern CSS features (Flexbox/Grid), crisp typography, smooth fade overlays, and a dark, earth-toned, sophisticated color palette.
*   **Responsive Web Design (RWD):** Fluid typography and media breakpoints optimized for viewports ranging from mobile displays to large desktop screens.
